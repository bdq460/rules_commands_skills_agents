# React 技术栈最佳实践

## 快速开始

- 使用 create-react-app 或 Vite 初始化项目
- 推荐使用 TypeScript

## 最佳实践

- 组件拆分与复用
- 使用函数式组件和 Hooks
- 状态管理推荐 Redux、Zustand
- 代码分割与懒加载
- 单元测试推荐 Jest + React Testing Library

## 常见问题

- 性能优化：使用 React.memo、useMemo、useCallback
- 路由管理：react-router
- 表单处理：formik、react-hook-form

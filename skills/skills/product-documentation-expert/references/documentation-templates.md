### 3. API文档

```markdown
# [产品名称] API 文档

## 概述

本文档描述了[产品名称]的RESTful API接口。

**Base URL**：`https://api.example.com/v1`

**认证方式**：Bearer Token

## 通用信息

### 认证

所有API请求都需要在HTTP Header中包含认证Token：

```http
Authorization: Bearer <your_token>

```

### 请求格式

Content-Type: `application/json`

### 响应格式

所有API响应都遵循以下格式：

**成功响应**：

```json
{
  "success": true,
  "data": {},
  "message": "操作成功"
}

```

**错误响应**：

```json
{
  "success": false,
  "error": {
    "code": "ERROR_CODE",
    "message": "错误描述",
    "details": {}
  }
}

```

### HTTP状态码

| 状态码 | 说明         |
| ------ | ------------ |
| 200    | 请求成功     |
| 201    | 创建成功     |
| 400    | 请求参数错误 |
| 401    | 未授权       |
| 403    | 禁止访问     |
| 404    | 资源不存在   |
| 429    | 请求过于频繁 |
| 500    | 服务器错误   |

### 分页

所有列表接口都支持分页参数：

| 参数     | 类型    | 必填 | 说明             |
| -------- | ------- | ---- | ---------------- |
| page     | integer | 否   | 页码，默认1      |
| pageSize | integer | 否   | 每页数量，默认10 |

**分页响应**：

```json
{
  "success": true,
  "data": [],
  "pagination": {
    "page": 1,
    "pageSize": 10,
    "total": 100,
    "totalPages": 10
  }
}

```

## 示例代码

### JavaScript/TypeScript

```typescript
import axios from "axios";

const api = axios.create({
  baseURL: "https://api.example.com/v1",
  headers: {
    "Content-Type": "application/json",
    Authorization: `Bearer ${token}`,
  },
});

// 用户登录
async function login(username: string, password: string) {
  const response = await api.post("/auth/login", {
    username,
    password,
  });
  return response.data;
}

// 创建订单
async function createOrder(items: any[], address: any) {
  const response = await api.post("/orders", {
    items,
    shippingAddress: address,
  });
  return response.data;
}

```

### Python

```python
import requests

BASE_URL = 'https://api.example.com/v1'
HEADERS = {
    'Content-Type': 'application/json',
    'Authorization': f'Bearer {token}'
}

# 用户登录
def login(username, password):
    response = requests.post(
        f'{BASE_URL}/auth/login',
        json={'username': username, 'password': password}
    )
    return response.json()

# 创建订单
def create_order(items, address):
    response = requests.post(
        f'{BASE_URL}/orders',
        headers=HEADERS,
        json={'items': items, 'shippingAddress': address}
    )
    return response.json()

```

### Q1: 如何注册账户？

**A**:

2. 点击右上角的"注册"按钮

3. 填写注册表单，包括：
   - 用户名（至少6个字符）
   - 邮箱地址
   - 密码（至少8个字符，包含字母和数字）

5. 检查您的邮箱，点击验证链接完成注册

**A**:

## 安全问题

### Q14: 如何保护我的账户安全？

**A**:

1. 使用强密码（至少8个字符，包含字母、数字和特殊字符）

2. 定期更改密码

3. 启用双因素认证

4. 不要在公共设备上保存密码

5. 不要点击可疑链接或下载未知附件

### Q15: 我发现可疑活动，该怎么办？

**A**:

1. 立即修改您的密码

2. 检查您的账户活动记录

3. 启用双因素认证（如果尚未启用）

4. 联系我们的[客服支持](support)报告可疑活动
